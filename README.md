# MinecraftClone (Java + LWJGL 3)

## Build & Run

Compile:

```java
mvn clean install
```

Run:

```java
mvn exec:java
```

## Controls

| Key          | Control               |
| ------------ | --------------------- |
| `Move Mouse` | look around           |
| `W`          | move forward          |
| `A`          | move left             |
| `S`          | move behind           |
| `D`          | move right            |
| `Space`      | move up               |
| `Left Shift` | move down             |
| `F2`         | take screenshot       |
| `F3`         | toggle wireframe mode |
